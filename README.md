# شبشب الهجاء

لعبة بسيطة: تضرب الصورة بالشبشب وتظهر جمل هجاء تتكرر بطرق مختلفة.
الإعداد الافتراضي ثابت على مقام النعل من الأعلى ووضع الهجاء بالتتابع.

## التشغيل في Codespace

```bash
npm start
```

ثم افتح الرابط:

```
http://localhost:8000
```

## تغيير الصورة

ضع صورتك داخل مجلد `assets` باسم واحد من التالي:

- `target.png`
- `target.jpg`
- `target.jpeg`
- `target.webp`

سيتم اختيار أول صورة يجدها التطبيق.

## تغيير الشبشب

ضع صورة الشبشب في مجلد `assets` باسم `slipper.png`.
إذا لم تتوفر الصورة، ستستمر اللعبة بدون عرض الشبشب.
